Odoo Leave Request Portal User Employee

This module allow you to employee(s) who are not real users of system but portal users / external user and it will allow to create leave request and edit leave request.

Main Features:

  • Allow you to portal user Leave Request.
  • Portal Users who are employee of system but not real users can fill/record Leave Request entry.
  • If your company using Leave Request application but not purchased real users from Odoo Enterprise then your employee can fill Leave Request as portal users.
  • No need to create real users in system if you are only using Leave Request module to make leave request entry for your all employees.
  • So you can create portal users and set it on employee form and employee can use that portal user logged to fill leave request.
  • Fully mobile responsive.
