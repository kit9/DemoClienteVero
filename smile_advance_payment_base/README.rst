=====================
Advance Payments Base
=====================

    :alt: Smile-SA/odoo_addons

|badge2| |badge3|

This module allows to configure:
    * Advance payment journals
    * Advance Accounts Payable & Receivable on partners

To test this module, you have to install
`smile_advance_payment_purchase <https://github.com/Smile-SA/odoo_addons/tree/11.0/smile_advance_payment_purchase>`_
.
