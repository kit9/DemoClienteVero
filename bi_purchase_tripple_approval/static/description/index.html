Purchase Tripple Approval Process

This app helps you to tripple approval process for Purchase Order while confirming Purchase order.

Key Features:

  • Purchase approval process: Purchase User -> Finance Manager -> Director
  • This App will add approval buttons on purchase order based on configurations in purchase setting. If order approval configure 5000 and purchase order is more then 5000 then it require approval from manager and then in second approval configure 8000 then if purchase order amount more then 8000 it will require second lavel approval from Director.
  • You can set three step verification approval process on Purchase Orders
  • You can set a limit amount on Purchase Order as Two and Three Step Verification
  • If Order's Total Amount exceeds the pre-defined limit,then Flow will be Two Step purchase validation process.
  • If Order's Total Amount exceeds the pre-defined limit of two step,then Flow will be Three Step validation process.

How To Use


Once module is installed Only User with "Finance Manager(Purchase)" groups can Confirm Purchase order from 'To Approve' state and User with " Director(Purchase)" groups can Confirm Purchase order from 'Second Approval' state .


Setup & Configure

Configure Approval Amounts into purchase settings.


Enable "Order Approval" and "Second Approval" to set Minimum Approval amount.

Three Step can be enabled only when Order Approval is Enabled

If Amount Total of Purchase Order will exceeds the limit of First Approval then flow will be Two Step.

If Amount Total of Purchase Order will exceeds the limit of Second Approval then flow will be Three Step.

Purchase User who don't have any Approval Group .


Order will be into To Approve state, because Total Amount of order is greater tha pre-defined First Approval Limit


Purchase Manager : Included into Finance Manager(Purchase) group


Order will be into Second Approval state, because Total Amount of order is greater tha pre-defined Second Approval Limit

Purchase Director : Included into Director(Purchase) Group


Order will be Confirmed state, because if Second Approval is Enabled , only user with Second Approval Group can Confirm Purchase Order
